This project comes with a FE to test the socket connection.
For this project to run, simply run `npm install` then `docker-compose up`

## Installation
$ npm install
$ docker-compose up


```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
